# InAmigos NGO Awareness Website

## About
This project is a simple NGO awareness webpage developed using HTML and CSS. It aims to spread awareness about InAmigos Foundation by showcasing its mission, ongoing projects, social impact, and volunteer opportunities, and connects visitors to the foundation's official website and donation page.

## Features
Responsive and clean design
NGO introduction
Ongoing projects section with real project photos from InAmigos Foundation
Social impact section
Working Call-to-Action buttons (Join Us / Donate Now)
Footer with contact details (address, email, phone)
Direct links to the official website (inamigosfoundation.org.in) and donation page

## Technologies Used
- HTML5
- CSS3
