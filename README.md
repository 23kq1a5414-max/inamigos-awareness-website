# InAmigos NGO Awareness Website

## About
This project is a simple NGO awareness webpage developed using HTML and CSS. It aims to spread awareness about InAmigos by showcasing its mission, ongoing projects, social impact, and volunteer opportunities.

## Features
- Responsive and clean design
- NGO introduction
- Ongoing projects section
- Social impact section
- Images and visuals
- Call-to-Action (Join Us)

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created as part of an internship task to demonstrate basic web development skills while promoting the work and initiatives of InAmigos.
